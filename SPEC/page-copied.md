# PAGE COPIED

## ЕСЛИ БУФЕР НЕ ПУСТОЙ

- Все блоки кроме информационного имеют верхний border

### Информационный блок
- Заполняет всё свободное пространство кроме других блоков, прижимая их вниз
- Заголовок "Copied!", крупный, центрованный, зелёный
- Подзаголовок:
   - Зависит от последнего действия
   - Если copied:
      - "Copied to clipboard: WHAT_COPIED"
      - WHAT_COPIED = то, что скопировано в clopboard
      - Если WHAT_COPIED != nothing, то чёрная, жирная
      - Если WHAT_COPIED = nothing, то серая (как fallback)
      - При сохранении в clopboard нового значения, WHAT_COPIED меняется соответственно
   - Если saved:
      - "Downloaded: WHAT_DOWNLOADED"
      - WHAT_DOWNLOADED = последний скопированный файл, чёрный, жирный
      - При сохранении файла, WHAT_COPIED меняется соответственно
   - Если ничего не скопировано и не сохранено, то строки нет
   - Наличие/отсутствие подзаголовка не должно смещать положения других элементов

### Copy text
- Всё inline
- Содержит:
   - "Copy text:"
   - Кнопки "Text", "Markdown"
- На кнопках есть иконки и название
- Выделенная кнопка:
   - Если окно только открылось, то выделенной кнопкой является та, которая соответствует сохранённым в clipboard данным
   - Далее выделенной яаляется последняя нажатая
   - Выделяется зелёным
- Связь с кэшем:
   - Если для какой-то из опций нечего отдать пользователю то кнопка серая, без изменения состояний
   - Учесть, что часть данных мы не храним в кэше, а вычисляем из других данных (которые в кэше)
- При нажатии на кнопку, значение копируется в буфер обмена

### Copy image
- Аналогично "Copy text:", но кнопки: "PNG", "JPEG"

### Download
- Кнопки:
   - Markdown
   - HTML
   - PNG
   - JPEG
- При нажатии на кнопку, скачивается соответствующий файл

### Developer tools
- Есть только если включено в настройках пользователя
- Каждая запись:
   - Отдельной строкой
   - Формат: [название в качестве label строки] [поле для копирования]
   - Строка заполняет всю ширину окна, без переносов
   - Поле для копирования:
      - Выглядит как код
      - В конце есть иконка copy
      - Если строка не вмещается, то обрезается визуально, но по факту она там полностью
      - Текст выделить нельзя
      - Все поля для копирвоания равной ширины
   - При нажатии на строку (включая label), значение копируется в буфер обмена

### Блок again
- Всего одна кнопка "START OVER", отцентрована
- Логика кнопки полностью совпадает с кнопкой "START" на странице START
