# Новая кнопка и логика

## ПО ШАГАМ
1. Переименования
    - Кнопку "START" переименовать в "COPY ELEMENT"
    - кнопку "START OVER" переименовать в "NEW ELEMENT"
2. Рядом с этими кнопками добавить кнопки
    - "COPY PAGE" (над  "COPY ELEMENT")
    - "NEW PAGE" (inline с "NEW ELEMENT")
3. Логика кнопки "COPY ELEMENT" и "NEW ELEMENT" остаётся неизменной
4. На "COPY PAGE" и "NEW PAGE" логика:
	1. Не запускать никаких режимов выделения
   2. Обработать весь `<html>...</html>` данной страницы полностью как если бы этот элемент был выбран пользователем

## ВАЖНО
- Задача максимально простая в плане изменений
- Минимальный фикс
